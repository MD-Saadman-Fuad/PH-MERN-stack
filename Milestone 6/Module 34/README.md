var is function-scoped and can be redeclared and updated. let and const are block-scoped. let can be updated but not redeclared in the same scope, while const cannot be updated or redeclared after assignment.

Answer:
map() returns a new array by applying a function to each element. forEach() executes a function for each element but does not return a new array. filter() returns a new array containing elements that pass a test provided by a function.

Arrow functions in ES6 are a concise way to write functions using the => syntax. They do not have their own this, arguments, or super, and are best for non-method functions.

Template literals in ES6 use backticks (`) and allow embedded expressions with ${}. They support multi-line strings and variable interpolation, making them more flexible than traditional string concatenation with +.
